# tikimiki

[cite_start]**Platforma za održavanje i učešće na hakatonima** [cite: 3]

[cite_start]Projekat "tikimiki" je razvijen u okviru predmeta Principi Softverskog Inženjerstva (SI3PSI) na Elektrotehničkom fakultetu Univerziteta u Beogradu[cite: 1, 24]. [cite_start]Platforma objedinjuje ceo životni ciklus hakatona na jednom mestu, od pronalaska događaja i formiranja tima, kroz aktivnu podršku tokom takmičenja, do karijernih ishoda koji iz njega mogu proisteći[cite: 43]. 

## 🛠️ Tehnologije
- [cite_start]**Frontend:** Next.js (uz HTML5, CSS3, AJAX) [cite: 177]
- [cite_start]**Backend:** Django [cite: 178]
- [cite_start]**Baza podataka:** PostgreSQL [cite: 178]

## ✨ Ključne funkcionalnosti
- [cite_start]**Pametno formiranje timova:** Automatsko popunjavanje profila na osnovu GitHub aktivnosti i AI sistem za predlaganje optimalnih timova na osnovu veština[cite: 44].
- [cite_start]**Ugrađena komunikacija:** Svaki hakaton na platformi dolazi sa ugrađenim serverom za komunikaciju, eliminišući potrebu za eksternim alatima poput Discord-a ili Slack-a[cite: 45].
- [cite_start]**Sistem uloga:** Platforma podržava Goste, Članove (sa ulogama takmičara ili moderatora), Organizacije i Administratore[cite: 50, 53, 59, 64, 70].
- [cite_start]**Sistem nagrađivanja:** Dodeljivanje poena i bedževa za različita postignuća nakon dostavljanja zvaničnih rezultata[cite: 133, 134].
- [cite_start]**Premium nalozi:** Mogućnost napredne personalizacije profila (animirani GIF-ovi, prilagođene boje) za isticanje u zajednici[cite: 62].
